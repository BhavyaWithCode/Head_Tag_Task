⭐⭐⭐Examples Of Head Tags⭐⭐⭐

💡What is head tags ?

➡️ Head tag means :- we can say that head tag is brain of the document because it gives data or information about that webpage.But It does'nt shows on a web page it helps the browser that how content should displayed on web page.

📚Types Of Tags and all examples are as under as follows:-

🌟Title Tag

🚀Title tags means:- It shows the title of the webpage in title bars or tab

🚀For example: Opening and closing tags of title and between content name of title.which show on tabs

🌟Meta Tag

🚀Meta tags means:- It provides a metadata about document

🚀For example: meta charset UTF-8
meta name = viewport content="width=device-width, initial-scale=1.0"

🌟Link Tag

🚀Link tags means:- Suppose you want to add any external link at that time link tags is been used.

🚀For example: link rel = "stylesheet" href = "style.css"

🌟Style Tag

🚀Style tags means:- It's Uses for internal source directly..

🚀For example: style body { background-color: #f1f1f1; font-family: Arial, sans-serif; } style

🌟Script Tag

🚀Script tags means:- you want to add any external script link at that time script tags is been used.

🚀For examplte: script src="script.js"
